# 고급 설정

## 플래그

Dashify Launcher를 CLI로 실행할 때 사용할 수 있는 플래그들이 있습니다.

### lang

:::note

인자는 'en-US' 또는 'ko-KR' 같은 형식이어야 합니다.

:::

표시할 언어를 고릅니다.

**값 타입:** `String`

### verbose

상세한 로그를 포함하여 출력할지 고릅니다.

**값 타입:** `Boolean`

### mcorigin

:::note

이 옵션은 구성 설정 파일에서 트래픽 모니터링을 켜야 유효합니다.

:::

마인크래프트 수신 포트를 변경합니다.

**값 타입:** `Integer`

### version

실행기의 버전을 출력할지 고릅니다.

**값 타입:** `Boolean`
